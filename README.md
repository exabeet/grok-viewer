Grok Viewer is a Chrome extension that adds a dedicated video viewer for Grok Imagine video generations.

It automatically collects and displays MP4 videos only, using your active Grok session.

## What it does
	• Displays all Grok Imagine videos (MP4 only) in a clean grid interface
	• Shows new videos instantly as soon as generation completes
	• Loads existing videos from Grok Favorites automatically
	• Lightbox video player with keyboard navigation
	• Repeat / Create Video directly from the viewer
	• You can generate your own videos inside the Extension
	• Download your videos in bulk or single
	• Delete your videos

## Installation:
**Clone or download the repository.**  
**Open Chrome and navigate to chrome://extensions.**  
**Enable Developer Mode.**  
**Click “Load unpacked” and select the project directory.**  
**Navigate to Grok Imagine to begin using the extension.**  

Scope and limitations
Grok Viewer is intended for personal use and operates only on content accessible to the logged-in user. The extension does not modify Grok content beyond user-initiated actions such as deletion.
